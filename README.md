# LocalLibrary++

Create a file called `.env` in the project root and copy the contents of [`.env.sample`](./.env.sample) to it.

```shell
docker compose up -d
```

```shell
docker exec -it local-library-pp-server-1 /bin/sh
```

```shell
npm install
```

```shell
npm run dev
```

Go to [localhost:3000](http://localhost:3000).
